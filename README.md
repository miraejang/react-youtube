# YouTube
React를 활용한 Youtube 클론 코딩입니다.    
　    
### 이 과정을 통해 얻은 것
- API를 공식문서를 읽고 사용할 수 있습니다.
- Postman을 사용할 수 있습니다.
- React Hook을 사용할 수 있습니다.
- PostCSS를 사용할 수 있습니다.    
　    
### 사용한 기술
- React
- React Hook
- axios
- Postman
- PostCSS    
　    
## 구현한 기능 설명
### 검색
검색한 내용에 해당되는 유튜브 동영상 리스트가 나옵니다.    
　    
### 동영상 상세페이지
동영상을 클릭하면 해당 동영상의 상세한 내용이 보여집니다.
